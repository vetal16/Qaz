Neural network demo - simple OCR

Demo instructions:

  • Draw a character (A, B or C) in the left square.
  • Click 'Read'. A downsampled version of the drawing appears in the middle square.
  • The network now tries to guess what you have drawn and displays the character in the right right square.
Training instructions:
The preloaded network only knows about the characters A, B and C. You can improve the recognition of these characters, or add new characters to your liking.
  • Type the character to train in the input field.
  • Draw the character in the left square.
  • Click 'Train character'.
  • Repeat this with different drawings of the same character to train the network.
